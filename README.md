# Sortowanie bąbelkowe

## Opis algorytmu 

Sortowanie bąbelkowe polega na wielokrotnym porrównywaniu sąsiednich elementów tablicy i zamienianiu ich miejscami, jeśli są w złej kolejności. Proces jest powtarzany tak długo, aż tablica zostanie posortowana

## Opis diagramu blokowego (tekstowy)

1. Start
2. Inicjalizacja zmiennej i = 0 oraz określenie tablicy n
3. Rozpoczęcie pętli zewnętrznej
4. Rozpoczęcie pętli wewnętrznej dla elementów od j = 0 do n - i - 2
5. Porównywanie elementów tab[j] i tab [j+1]
6. Jeśli tab[j] > tab[j+1], następuje zmiana elementów
7. Zwiększenie indeksu i kontynuacja pętli wewnętrznej
8. Zwiększenie indeksu i kontynuacja pętli zewnętrznej
9. Koniec algorytmu
